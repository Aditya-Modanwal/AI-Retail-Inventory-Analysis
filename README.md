# AI Retail Inventory Health Analysis

This project analyzes retail sales data using Machine Learning to help businesses understand demand patterns and inventory health.

## Features
- Sales data analysis
- Demand prediction using Random Forest
- Inventory coverage calculation
- Overstock and stockout detection
- Web dashboard using Flask

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Flask
- Matplotlib

## Dataset
Supermarket Sales Dataset containing 1000 retail transactions.

## Purpose
To demonstrate how AI and data analytics can support retail inventory decision making.
